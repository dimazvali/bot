# Сайт фотографа — аэрофотография

**Дата:** 2026-06-05  
**Статус:** Approved

## Суть

Лёгкий монохромный портфолио-сайт для аэрофотографа. Основной контент — горизонтальные снимки с коптера. Сайт встраивается в существующий Node.js/Express проект как отдельный view (Pug).

---

## Структура

### Страницы

1. **Галерея** (`/photo` или `/aero`) — основная страница, сайдбар + masonry-сетка
2. **Снимок** (`/photo/:country/:series/:id`) — страница одного фото
3. **О себе** (`/photo/about`) — минимальный текст + контакты

### Навигация (сайдбар)

Двухуровневая: **Страна → Сюжет (серия)**

- Страны раскрываются аккордеоном
- Текущий активный сюжет подсвечен
- В нижней части сайдбара — ссылка «О себе»

Пример структуры URL: `/photo/georgia/tbilisi`

---

## Визуальный дизайн

### Монохром + переключатель тем

Два варианта — тёмная и светлая, переключатель в шапке. CSS-переменные для цветов, смена через `data-theme` атрибут на `<html>`.

**Тёмная тема:** фон `#0d0d0d`, сайдбар `#0a0a0a`, текст `#ccc`, акценты `#fff`  
**Светлая тема:** фон `#f5f5f5`, сайдбар `#fff`, текст `#333`, акценты `#000`

### Типографика

- Логотип/заголовки: monospace, letter-spacing 3-4px, caps
- Навигация: 9px, letter-spacing 2px, caps
- Описания на странице фото: sans-serif, 13px

### Сайдбар

- Ширина: 160px, фиксированный (не схлопывается)
- Логотип вверху
- Аккордеон стран со стрелкой ▸/▾
- Сюжеты с отступом под раскрытой страной

---

## Галерея

- **Masonry** в 3 колонки (CSS `columns`)
- Все фото горизонтальные (aspect-ratio 16/9 или свободный)
- При наведении: тёмный оверлей + название + год/локация
- Клик на фото → страница снимка
- Счётчик фото в серии: `ГРУЗИЯ · ТБИЛИСИ · 24 фото`

---

## Страница снимка

- Фото на всю ширину, aspect-ratio 16/7 (широкий формат)
- Хлебная крошка / кнопка назад: `← ТБИЛИСИ`
- Заголовок + локация + дата (monospace caps)
- Короткое текстовое описание (необязательно)
- Навигация ← ПРЕД. / СЛЕД. → внутри серии

---

## Данные

Фотографии хранятся в `public/images/photo/` по структуре:  
`public/images/photo/{country}/{series}/{filename}.jpg`

Метаданные — `data/photo.json` в корне проекта:  
```json
{
  "georgia": {
    "label": "Грузия",
    "series": {
      "tbilisi": {
        "label": "Тбилиси",
        "photos": [
          { "file": "metelhi-dawn.jpg", "title": "Метехи на рассвете", "date": "Май 2024", "desc": "..." }
        ]
      }
    }
  }
}
```

## Технический стек

- **Backend:** Node.js + Express (существующий проект)
- **Шаблоны:** Pug (как весь проект)
- **Стили:** один CSS-файл `public/stylesheets/photo.css`
- **Роутер:** `routes/photo.js` → подключается в `app.js`
- **JS на клиенте:** минимум — только переключатель тем + аккордеон (vanilla JS, без фреймворков)
